# Passport.js Practice

## Skills used

- Passport.js
- Typescript
- Mongodb atlas

## Providers

- Google
- Facebook

## Configuration

1.  Clone the repo
2.  Make a `keys.ts` file under the `config/`. It should look like `keys.ts.example`.
3.  Install dependencies and Run
    ```
    yarn
    yarn start
    ```
    Should be running on http://localhost:3000
